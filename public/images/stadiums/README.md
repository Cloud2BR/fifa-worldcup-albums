# Stadium photographs

Drop free-license stadium images here. Each file must match the `file`
field of an entry in `src/data/stadiumImages.json`, e.g.
`maracana.jpg`.

Only public-domain, CC0, CC-BY, or CC-BY-SA images are accepted, and
every file must be listed with its author, license, and source URL in
the root-level `CREDITS.md`. Until a file is present, the `VirtualAlbum`
component falls back to its built-in SVG stadium silhouette
automatically.
