# references

Reference-only images used for verification and research.

## Folders

- `maps/`: Satellite-style map captures.
- `place-photos/`: Place hero-photo style captures (closest to real venue photo examples).
- `streetview/`: Reserved for Street View captures when available.

## Notes

- These are not primary runtime assets unless explicitly wired into UI.
- Keep attribution and source links in `src/data/mapSatelliteReferences.json` and `src/data/mapPhotoReferences.json`.
- If Street View is unavailable for a venue, keep that venue in `place-photos/` instead.
